🌙 Lash Lunar – Website
📌 Overview

Lash Lunar is a modern, responsive landing page designed for a lash and brow studio based in Cape Town, South Africa. The website showcases services, pricing, policies, and contact details while maintaining a soft, minimal, and luxurious aesthetic.

The goal of this project was to create a visually appealing and user-friendly experience that reflects the brand’s identity while ensuring smooth navigation across all devices.

✨ Features
Responsive Design
Fully optimized for mobile, tablet, and desktop screens.
Smooth Navigation
Scroll-based navigation with anchored sections for easy access.
Mobile Menu Overlay
Clean, accessible mobile navigation with toggle functionality.
Services & Pricing Section
Structured cards displaying lash and brow services with pricing.
Expandable Sections
Interactive toggles for:
Services
Booking & Policies
Gift Vouchers
Client Reviews
Styled testimonial cards to build trust and credibility.
Contact Section
Multiple contact options including:
Email
WhatsApp
Instagram
TikTok
Downloadable Resource
Pre & Post Care guide available as a downloadable PDF.
🎨 Design Approach

The design focuses on:

Soft, neutral tones for a calming and elegant feel
Clean typography with strong spacing for readability
Subtle hover effects and shadows for interactivity
A balance between minimalism and visual interest

The layout was intentionally kept simple and structured to highlight content without overwhelming the user.

🛠️ Technologies Used
HTML5 – Structure and content
Tailwind CSS – Styling and responsive design
JavaScript (ES Modules) – Interactivity (menu toggle, section toggles)
CSS (Custom styles) – Additional refinements
